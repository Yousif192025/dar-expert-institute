# دليل إعداد Formspree للنماذج

## الخطوات (5 دقائق فقط)

### 1. أنشئ حساباً مجانياً
- اذهب لـ https://formspree.io
- سجّل بالإيميل المطلوب
- الخطة المجانية: 50 رسالة/شهر

### 2. أنشئ نموذجاً جديداً
- اضغط "New Form"
- أدخل البريد الإلكتروني الذي تريد استقبال الرسائل عليه
- انسخ الـ Form ID (مثال: `xpzgkwna`)

### 3. استبدل في جميع ملفات HTML
ابحث عن النص التالي في كل الملفات:
```
YOUR_FORM_ID
```
واستبدله بالـ ID الحقيقي (مثال: `xpzgkwna`)

### الملفات التي تحتوي على النماذج:
- `index.html` — نموذج التسجيل الرئيسي
- `contact.html` — نموذج التواصل  
- `corporate.html` — نموذج الطلب المؤسسي
- `programs/*.html` — نموذج التسجيل في البرامج

### 4. اختبر النموذج
بعد الرفع على GitHub، اختبر إرسال رسالة من الموقع.
ستصل رسالة تأكيد لإيميلك.

## ملاحظات
- Formspree يُرسل أيضاً إيميل تأكيد للمستخدم إذا أضفت:
  `<input type="hidden" name="_replyto" value="...">` في النموذج
- يمكن تخصيص رسالة الشكر عبر: `_next: thank-you.html`
- للخصوصية: أضف `<input type="hidden" name="_subject" value="تسجيل جديد">`
